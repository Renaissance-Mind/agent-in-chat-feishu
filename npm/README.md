# agent-in-chat-feishu

Run local AI coding agents inside Feishu/Lark chat loops.

## Install

```bash
npm install -g @renaissancemind/agent-in-chat-feishu
```

The installed CLI command is:

```bash
agentchat
```

## Quick Start

```bash
agentchat feishu setup --project my-project
agentchat
```
